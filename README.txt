== Description ==

Product Quick View for WooCommerce gives gives the non-developer an easy solution to adding a quixk view button to products on your store.  When the user clicks the button, instanatly they are greeted with a popup showing off the features of the product, an advanced product gallery and the ability to see the product or add to cart.  This plugin not only gives you the ability to add a button, you also get shortcode so you can add the button anywhere on your website. In addition to shortcode this plugin comes equiped with developer hooks to replace the standard WooCommerce gallery on the single product page.



A few notes about the sections above:

*   Adds a quick view button to WooCommerce procuct listings
*   Imporoved product gallery available
*   Hide quick view button on certain products

 

== Installation ==


= From your WordPress dashboard = 
Visit Plugins > Add New
Search for ‘Product Quick View for WooCommerce’
Activate Prodduct  Quick View for WooCommerce from your Plugins page. 
The Quick View Button will now show on all product listings


= Install Manually =
Download Product Quick View for WooCommerce from https://gitlab.com/greatdane89/woocommerce-quick-view.
Upload the Zipped folder to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)
Activate bbPress from your Plugins page.
The Quick View Button will now show on all product listings


= Extra = 
Visit  Woocommerce -> Quick View -> settings and adjust your configuration.
Visit  Woocommerce -> Quick View -> advanced settings for developer settings such as conditional scripts and defer/async plugin loaded dependencies

== Frequently Asked Questions ==

= Can I hide the quick view button on certain products? =

Yes!!! simply go to edit product -> Product Data -> Product Quick View and check the Hide Quick View box.

= Can I change the quick view text? =

You can change the text of the quick view button globally by going to Woocommerce -> Quick View -> settings and editing the "Button text:" input box.

= Can I hide the button on all products? = 

Yes, if you go to Woocommerce -> Quick View -> settings there is a checkbox that will allow you to hide the button globally

= Can I place the quick view button somwhere else on the website? =

Yes, with the correct hooks you can place the quick view button anywhare as long as it has a product id passed in or the hook is within a product loop.
